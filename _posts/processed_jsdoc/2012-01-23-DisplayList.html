DisplayList

The DisplayList mixin, a mixin for display objects that may contain other display objects.

Methods

<static> addChild

Adds a child at the end list of contained display objects.

If an index is given, the child is inserted at that index, moving array items at that index and onwards one along. If the child is inserted at a different point (different parent and/or index), it is removed first.

Parameters:
Name Type Argument Description
child DisplayObject

The child to add
index number <optional>

The index for the child in the DisplayList
Returns:
